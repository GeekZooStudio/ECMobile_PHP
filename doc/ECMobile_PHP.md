# 后台配置

1. 下载源代码并解压
2. 搜索源代码，将所有Your Information替换为您的网站信息
3. 将/ECMobile/目录，复制到ecshop网站的根目录
4. 对/ECMobile/目录，进行用户组及读写权限设置
5. 尝试在浏览器中访问 `http://<您的网站>/ECMobile`，将看到欢迎信息。
6. 接下来，您可以进行API测试。

# 客户端配置

1. 打开iOS/Android工程
2. 根据文档将API URL替换为 http://<您的网站>/ECMobile/?url=
3. 打包安装，并进行测试
